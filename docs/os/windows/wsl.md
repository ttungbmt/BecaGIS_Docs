## Installation

Check the WSL mode

```powershell
wsl.exe -l -v
```

```powershell
Install-Module Alt3.Docusaurus.Powershell -Scope CurrentUser
Install-Module PlatyPS -Scope CurrentUser
```

Change your default WSL distro

```powershell
wsl --set-default <distro name>
```
